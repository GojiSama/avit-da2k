# avit-da2k

> oh-my-zsh theme based on avit theme

## Dependencies

```
wget
```

## Installation

- Copy `avit-da2k.zsh-theme` to `~/.oh-my-zsh/themes/`
- Add `ZSH_THEME="avit-da2k"` on `~/.zshrc`
- Run `source ~/.zshrc`

Enjoy ;)
