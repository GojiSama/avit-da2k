# avit-da2k

> oh-my-zsh theme based on avit theme

## Installation

- Copy [`avit-da2k.zsh-theme`](https://raw.githubusercontent.com/fdaciuk/avit-da2k/master/avit-da2k.zsh-theme) to `~/.oh-my-zsh/themes/`
- Add `ZSH_THEME="avit-da2k"` on `~/.zshrc`
- Run `source ~/.zshrc`

Enjoy ;)
